# 图片目录说明

## 个人照片

请将您的个人照片放置在此目录下，文件名为：`profile.jpg`

### 图片要求：
- 文件名：`profile.jpg`（或 `.png` 格式）
- 建议尺寸：至少 600x600 像素
- 格式：JPG 或 PNG
- 建议：正方形照片，清晰度较高

### 原始图片路径
如果您有原始图片文件（路径：`tmp/211602429442_docxword_media_image1.jpeg`），请：
1. 将图片复制到此目录
2. 重命名为 `profile.jpg`
3. 或者修改 `index.html` 中的图片路径

### 如何修改图片路径

如果您的图片文件名不同，可以在 `index.html` 中修改：

```html
<img src="images/profile.jpg" alt="王明宇" class="profile-image">
```

将 `images/profile.jpg` 改为您的实际图片路径。
